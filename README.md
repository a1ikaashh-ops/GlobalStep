# GlobalStep

GlobalStep — образовательная платформа для обучения за границей.

Проект реализован на React (Vite) с использованием React Router. Все тексты на русском языке.

Особенности:
- Страницы: Главная, Программы, Университеты, Требования, О нас, Контакты
- Реалистичные данные программ и университетов в `src/data`
- Адаптивный дизайн, брендовые цвета: #3B82F6 и #0F172A
- Hover-эффекты и аккуратные анимации

Как запустить (macOS, zsh):

```bash
# в корне проекта
npm install
npm run dev
```

Откройте http://localhost:5173

Примечание: проект подготовлен для демонстрации. Форма контактов пока отправляет данные на клиенте (console.log). Для продакшн интегрируйте бэкенд или почтовый сервис.

Структура:
- src/components — переиспользуемые компоненты
- src/pages — страницы маршрутов
- src/data — примеры программ и университетов
- src/index.css — глобальные стили
